User Activity Logged

For the purpose of this assignment, I have logged the following actions: User upvoting an answer, User opening the link of a question, User clicking on a tag(eg: java, css) to access all questions under that tag , User clicking on a StackOverflow recommended job link and user copying any text/answer from the Stackoverflow webpage.

1.Upvoting an answer: A user upvoting an answer will definitely mean the answer was useful to him, also that the user has gained an aditional understanding of the topic associated with that particular Q&A

2.User clicking on a question to open the webpage having all its answers. This absolutely means the user is interested in this topic and it is a topic he most likely wishes to learn more about.

3.User clicking on a tag(for instance, java, spring etc).This can be understood to mean, the user would like to see more questions related to that topic. It also means that perhaps, the user has gained sufficient understanding of the topic to explore and seek out questions he may be able to answer

4.User clicking on a StackOverflow job link: Since SO displays job roles prominently these days and most of the times it is based on the technologies which we explore actively on StackOverflow, I feel that tracking the job roles ,the user is looking at might give us insight into the technologies he wishes to learn more about and the topics that motivate him

5. User copying answers from the webpage: If a user is copying text from the answers, he is most probably doing so for easy reference later. It might represent a topic the user is facing difficulty with as he needs to copy the text. It might also mean topics he is very actively using in his daily work or study.
